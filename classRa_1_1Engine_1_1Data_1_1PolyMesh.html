Detailed Description

PolyMesh, own a Core::Geometry::PolyMesh This class handle the GPU representation of a polyhedron mesh. Each face of the polyhedron (typically quads) are assume to be planar and convex. Simple triangulation is performed on the fly before sending data to the GPU.
