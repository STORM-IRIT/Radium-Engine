using Ra::Engine::Rendering::EngineRenderTechniques::DefaultTechniqueBuilder = typedef std::function<void( RenderTechnique& rt, bool isTransparent )>

A default technique builder is a function that will fill the given RenderTechnique with the default configurations associated with a material This function will take as param the render technique to fill and a boolean set to true if the material might be transparent under some parameter values
