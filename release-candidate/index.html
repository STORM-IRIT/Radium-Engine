Radium Engine

Radium is a research 3D Engine for rendering, animation and processing. Developed and maintained by the STORM research group.

The source code is hosted on github: https://github.com/STORM-IRIT/Radium-Engine

Overview

See this presentation for an overview of the project.

The documentation is structured as follow:

  • Radium Basics : how to compile Radium and use it in your own project.
  • Developer manual : general topics (e.g. coding conventions) and description of the concepts found in the API of Radium.
  • Radium Concepts : high level concepts used in Radium Engine
