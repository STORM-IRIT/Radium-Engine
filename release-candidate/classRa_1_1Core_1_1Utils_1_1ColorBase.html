Detailed Description

template<typename _Scalar>
class Ra::Core::Utils::ColorBase< _Scalar >

Colors are defined as vector4, i.e. 4 Scalars in RGBA order. displayable colors should have all their coordinates between 0 and 1.

Warning
Vector arithmetics can be used to add, substract, multiply or divide colors with scalars and other colors. In that case, alpha is treated as any other component. It is fine for most cases, however adding two colors (e.g. Blue()+Red()) may lead to unconsistent alpha: Blue()+Red() = {1, 0, 1, 2}. In that case, only the rgb values need to be added:
Color result = Color::fromRGB( Blue().rgb()+Red().rgb() ); // result = {1, 0, 1, 1}
Ra::Core::Utils::ColorBase< Scalar >
The validity of alpha can be checked using hasValidAlpha().
