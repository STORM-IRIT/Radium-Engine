Detailed Description

template<typename T>
class Ra::Dataflow::Core::Sources::SingleDataSourceNode< T >

Base class for nodes that will give access to some input data to the graph. This class can be used to feed nodes on a dataflow graph with some data coming from outside the graph or from the source node itself.

The data delivered by the node can be explicitly set/get or can be made editable.
