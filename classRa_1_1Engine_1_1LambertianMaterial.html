Detailed Description

Implementation of the Lambertian Material BSDF. This material implements a lambertian diffuse BSDF. This material could not be loaded from a file and must be defined and associated to renderobjects programatically.

This Material is parameterized either globally, by setting the m_color base color or locally, by setting a vertex color attribute that define the per vertex color and setting m_perVertexColor to true.
